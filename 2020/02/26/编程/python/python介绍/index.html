所以接下来只要对每一位数字化一下是否可以通过2或者 5 得到

最后求一下2和5分别有多少个，取最小的那个就可以了 因为2和5都是素数，所以不存在取得的数重复现象

总结一下：求每一个数可以得到几个2和几个5
