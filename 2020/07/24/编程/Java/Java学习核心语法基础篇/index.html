2.2 常量定义

3标识符命名规范

  • 在java 中能够自定义的字符串都是标识符。
  • 以字母、数字、下划线 、$ 开头。
  • 命名采用驼峰命名方式，并且不能使用java中的保留关键字。
  • 标识符命名有意义，不要使用中文、拼音缩写命名。
  • 变量和方法名称必须使用小写字母开始；类名必须使用大写字母开头；常量必须全部大写，多个单词使用下划线分割。
